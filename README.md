# Setup Mac

## Usage

1. Apple ID for mas-cli.

```bash
$ cp provisioning/host_vars/localhost.example provisioning/host_vars/localhost
$ vi provisioning/host_vars/localhost
```

2. Execute

```bash
$ make
```
